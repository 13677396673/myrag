### 1.2 依赖流向图

```
完全独立层 (无外部依赖):
  M01 Config        M02 Models        M03 Schemas        M08 Interfaces

半独立层 (依赖 Config 或 Interfaces):
  M04 Database ──── M05 Security ──── M06 Storage ──── M07 TaskQueue
  M09 Parsers ───── M10 Splitters ─── M11 Embedding ── M12 VectorStore
  M13 LLMs ──────── M14 Retrievers

编排层 (组合多个模块):
  M15 Pipeline ──── M16 RAG Engine

业务服务层 (依赖 Database + 编排层):
  M17 UserService  M18 DatasetService  M19 DocumentService
  M20 ConvService  M21 AdminService

API + 组装层:
  M22 API Routes ──→ M23 Container (组装所有)
```

**设计原则**：
- 箭头方向 = 依赖方向，箭头指向的模块被依赖
- 同一层的模块之间**禁止相互依赖**
- 上层可以依赖下层，下层不能依赖上层
- M02 Models 和 M03 Schemas 是纯数据结构，被所有业务模块引用

## 二、模块间解耦与依赖注入原则

### 2.1 依赖注入方式

本项目采用**手动构造函数注入**，核心规则：

```python
# ✅ 正确：依赖通过构造函数注入
class DocumentService:
    def __init__(self, db: DatabaseSession, storage: FileStorageBackend, task_queue: TaskQueueBackend):
        self._db = db
        self._storage = storage
        self._task_queue = task_queue

# ❌ 错误：模块内部自行创建依赖
class DocumentService:
    def __init__(self):
        self._db = get_database()  # 禁止

# ❌ 错误：使用全局变量
db = Session()  # 禁止

# ❌ 错误：使用类变量
class DocumentService:
    db = Session()  # 禁止
```

### 2.2 模块的"接口 + 实现"分离

每个可替换的模块（M06 Storage、M07 TaskQueue、M09-M13）遵循：

```
rag/
└── parsers/                    # 模块名
    ├── __init__.py             # 导出 register_parsers() 和 get_parser()
    ├── base.py                 # 如果该模块有自己的内部基类（可选）
    ├── text_parser.py          # 实现 1
    ├── pdf_parser.py           # 实现 2
    ├── docx_parser.py          # 实现 3
    └── tests/                  # 本模块的独立测试
        ├── __init__.py
        ├── fixtures/           # 本模块的测试夹具
        │   ├── sample.txt
        │   └── sample.pdf
        └── test_text_parser.py
```

### 2.3 模块的公共接口导出规范

每个模块的 `__init__.py` 遵循统一模式：

```python
# rag/parsers/__init__.py

from .text_parser import TextParser
from .pdf_parser import PDFParser
from .docx_parser import DocxParser
from .parser_router import ParserRouter

def register_parsers(router: ParserRouter) -> None:
    """注册所有内置解析器到解析器路由"""
    router.register(TextParser)
    router.register(PDFParser)
    router.register(DocxParser)

__all__ = [
    "TextParser",
    "PDFParser",
    "DocxParser",
    "ParserRouter",
    "register_parsers",
]
```

### 2.4 模块异常设计规范

每个模块定义自己的异常类，继承自 `RagError` 基类：

```python
# core/exceptions.py
class RagError(Exception):
    """所有模块异常的基类"""
    def __init__(self, message: str, code: str = "INTERNAL_ERROR", detail: dict = None):
        self.message = message
        self.code = code
        self.detail = detail or {}
        super().__init__(self.message)

class ConfigError(RagError):
    """配置相关错误"""
    pass

class StorageError(RagError):
    """文件存储相关错误"""
    pass

# ... 每个模块有自己的异常
```

### Module-01: 配置管理模块 (config)

**路径**: `backend/app/config/`

**独立性**: ⭐⭐⭐ 完全独立。不依赖任何其他模块，纯数据类。

**职责**：
- 从 `.env` 文件、环境变量、`config.yaml` 中分层加载配置
- 提供类型安全的配置访问（通过 pydantic-settings）
- 所有配置项集中一处管理

#### 类设计


### Module-02: ORM 模型模块 (models)

**路径**: `backend/app/models/`

**独立性**: ⭐⭐⭐ 完全独立。纯 SQLAlchemy ORM 定义，无业务逻辑。

**职责**：
- 定义所有数据库表对应的 ORM 模型
- 定义表之间的关系（外键、关联表）
- 不包含任何业务方法

#### 类设计

```python
# models/base.py
from sqlalchemy.orm import DeclarativeBase
from sqlalchemy import Column, DateTime, func
from datetime import datetime
import uuid

class Base(DeclarativeBase):
    """所有 ORM 模型的基类"""
    pass

def generate_uuid() -> str:
    return str(uuid.uuid4())


# models/mixins.py
class TimestampMixin:
    """自动时间戳混入"""
    created_at = Column(DateTime, nullable=False, server_default=func.now())
    updated_at = Column(DateTime, nullable=False, server_default=func.now(), onupdate=func.now())
```

```python
# models/user.py
from sqlalchemy import Column, String, Boolean
from .base import Base, generate_uuid, TimestampMixin

class User(Base, TimestampMixin):
    __tablename__ = "users"

    id = Column(String(36), primary_key=True, default=generate_uuid)
    username = Column(String(50), unique=True, nullable=False, index=True)
    email = Column(String(255), unique=True, nullable=False, index=True)
    password_hash = Column(String(255), nullable=False)
    role = Column(String(20), nullable=False, default="user")  # user | admin
    is_active = Column(Boolean, default=True)
```

```python
# models/dataset.py
from sqlalchemy import Column, String, Text, ForeignKey
from sqlalchemy.orm import relationship
from .base import Base, generate_uuid, TimestampMixin

class Dataset(Base, TimestampMixin):
```

### Module-03: Pydantic 模式模块 (schemas)

**路径**: `backend/app/schemas/`

**独立性**: ⭐⭐⭐ 完全独立。纯 Pydantic 请求/响应模型定义。

```python
from sqlalchemy.pool import NullPool

class DatabaseManager:

        self._engine = None
        self._session_factory = None

    def get_session(self) -> AsyncSession:
        """获取一个新的数据库会话"""
        return self._session_factory()

    @property
    def session_factory(self) -> async_sessionmaker[AsyncSession]:
```

```python
# core/security.py

import bcrypt
import jwt
from datetime import datetime, timedelta, timezone
from typing import Optional
from app.config.settings import Settings
from app.schemas.user import TokenResponse, UserResponse

class SecurityManager:
    """安全认证管理器"""

    def __init__(self, settings: Settings):
        self._settings = settings

    # ── 密码处理 ──
    def hash_password(self, password: str) -> str:
        """对密码进行 bcrypt 哈希"""
        salt = bcrypt.gensalt()
        return bcrypt.hashpw(password.encode("utf-8"), salt).decode("utf-8")

    def verify_password(self, plain_password: str, hashed_password: str) -> bool:
        """验证密码是否匹配哈希"""
        return bcrypt.checkpw(
            plain_password.encode("utf-8"),
            hashed_password.encode("utf-8"),
        )

    # ── JWT 处理 ──
    def create_access_token(self, user_id: str, role: str) -> str:
        """创建 JWT Token"""
        expire = datetime.now(timezone.utc) + timedelta(
            minutes=self._settings.JWT_ACCESS_TOKEN_EXPIRE_MINUTES
        )
        payload = {
            "sub": user_id,
            "role": role,
            "exp": expire,
            "iat": datetime.now(timezone.utc),
        }
        return jwt.encode(
            payload,
            self._settings.JWT_SECRET_KEY,
            algorithm=self._settings.JWT_ALGORITHM,
        )

    def verify_token(self, token: str) -> Optional[dict]:
        """验证 JWT Token，返回 payload 或 None"""
        try:
            payload = jwt.decode(
                token,
                self._settings.JWT_SECRET_KEY,
                algorithms=[self._settings.JWT_ALGORITHM],
            )
            return payload
        except jwt.ExpiredSignatureError:
            return None
        except jwt.InvalidTokenError:
            return None
```

```python
# core/storage/local_storage.py
import aiofiles
from pathlib import Path
from .base import FileStorageBackend

class LocalFileStorage(FileStorageBackend):
    """本地文件系统存储实现"""

    def __init__(self, base_path: str):
        self._base_path = Path(base_path)
        self._base_path.mkdir(parents=True, exist_ok=True)

    async def save(self, file_path: str, content: bytes) -> str:
        full_path = self._base_path / file_path
        full_path.parent.mkdir(parents=True, exist_ok=True)
        async with aiofiles.open(full_path, "wb") as f:
            await f.write(content)
        return str(full_path)

    async def read(self, storage_path: str) -> Optional[bytes]:
        full_path = Path(storage_path)
        if not full_path.exists():
            return None
        async with aiofiles.open(full_path, "rb") as f:
            return await f.read()

    async def delete(self, storage_path: str) -> bool:
        full_path = Path(storage_path)
        if full_path.exists():
            full_path.unlink()
            return True
        return False

    async def exists(self, storage_path: str) -> bool:
        return Path(storage_path).exists()
```

```python
# core/task_queue/huey_queue.py
from huey import SqliteHuey
from huey.api import Task as HueyTask
from .base import TaskQueueBackend
from typing import Callable, Any, Optional
import tempfile
import os

class HueyTaskQueue(TaskQueueBackend):
    """基于 Huey + SQLite 的任务队列实现（本地默认）"""

    def __init__(self, db_path: str = "./data/tasks.sqlite"):
        self._huey = SqliteHuey(name="myrag", filename=db_path)
        self._tasks: dict[str, Callable] = {}

    def register_task(self, name: str, fn: Callable):
        """注册任务函数"""
        @self._huey.task(name=name)
        def _wrapped(*args, **kwargs):
            return fn(*args, **kwargs)
        self._tasks[name] = _wrapped

    def enqueue(self, task_name: str, *args, **kwargs) -> str:
        if task_name not in self._tasks:
            raise ValueError(f"未注册的任务: {task_name}")
        result = self._tasks[task_name](*args, **kwargs)
        return str(result.id)

    def enqueue_with_delay(self, task_name: str, delay_seconds: int,
                           *args, **kwargs) -> str:
        if task_name not in self._tasks:
            raise ValueError(f"未注册的任务: {task_name}")
        result = self._tasks[task_name].schedule(
            args=args,
            kwargs=kwargs,
            delay=delay_seconds,
        )
        return str(result.id)

    def get_task_status(self, task_id: str) -> str:
        try:
            task = HueyTask.from_id(self._huey, task_id)
            if task is None:
                return "pending"
            # Huey 状态: waiting -> running -> completed
            return task.status
        except Exception:
            return "failed"

    def get_result(self, task_id: str, timeout: int = 10) -> Any:
        task = HueyTask.from_id(self._huey, task_id)
        if task:
            return task.get(timeout=timeout)
        return None
```

```python
        self._registry: Dict[str, Type[DocumentParser]] = {}

    def register(self, parser_cls: Type[DocumentParser]) -> None:
        """注册解析器"""
        for ext in parser_cls.supported_extensions():
            self._registry[ext.lower()] = parser_cls

    def get_parser(self, extension: str) -> Optional[DocumentParser]:
        """根据扩展名获取解析器实例"""
        parser_cls = self._registry.get(extension.lower())
        if parser_cls is None:
            return None
        return parser_cls()

    def get_supported_extensions(self) -> List[str]:
        """返回所有支持的扩展名"""
        return list(self._registry.keys())
```

```python
# rag/splitters/fixed_splitter.py
from app.rag.interfaces.splitter import TextSplitter, DocumentChunk
from typing import List, Optional

class FixedSizeSplitter(TextSplitter):
    """固定大小切片器：按 chunk_size 切分，支持 overlap"""

    def __init__(self, chunk_size: int = 512, chunk_overlap: int = 64):
        if chunk_overlap >= chunk_size:
            raise ValueError(f"chunk_overlap ({chunk_overlap}) 必须小于 chunk_size ({chunk_size})")
        self._chunk_size = chunk_size
        self._chunk_overlap = chunk_overlap

    def split(self, text: str, metadata: Optional[dict] = None) -> List[DocumentChunk]:
        chunks = []
        metadata = metadata or {}
        start = 0
        text_len = len(text)
        index = 0

        while start < text_len:
            end = start + self._chunk_size
            chunk_text = text[start:end]
            chunks.append(DocumentChunk(
                content=chunk_text,
                chunk_index=index,
                metadata={**metadata, "chunk_size": len(chunk_text)},
            ))
            index += 1
            if end >= text_len:
                break
            start = end - self._chunk_overlap

        return chunks

    @property
    def type_name(self) -> str:
        return "fixed"
```

```python
import chromadb
from chromadb.config import Settings as ChromaSettings
from app.rag.interfaces.vector_store import VectorStore, SearchResult
from typing import List, Optional, Dict, Any

class ChromaDBStore(VectorStore):
    """ChromaDB 向量存储实现"""

    def __init__(self, persist_directory: str = "./data/chromadb",
                 collection_name: str = "documents"):
        self._client = chromadb.PersistentClient(
            path=persist_directory,
            settings=ChromaSettings(anonymized_telemetry=False),
        )
        self._collection = self._client.get_or_create_collection(
            name=collection_name,
            metadata={"hnsw:space": "cosine"},
        )

    def add_embeddings(self, ids: List[str], vectors: List[List[float]],
                       metadatas: List[Dict[str, Any]]) -> None:
        # ChromaDB 要求 metadata 值必须为基本类型（str/int/float/bool）
        sanitized = []
        for m in metadatas:
            sanitized.append({
                k: (str(v) if isinstance(v, (dict, list)) else v)
                for k, v in m.items()
            })
        self._collection.add(
            ids=ids,
            embeddings=vectors,
            metadatas=sanitized,
        )

    def search(self, query_vector: List[float], top_k: int = 5,
               filter_conditions: Optional[Dict[str, Any]] = None) -> List[SearchResult]:
        results = self._collection.query(
            query_embeddings=[query_vector],
            n_results=top_k,
            where=filter_conditions,
        )
        if not results["ids"]:
            return []
        items = []
        for i in range(len(results["ids"][0])):
            items.append(SearchResult(
                id=results["ids"][0][i],
                score=results["distances"][0][i] if results.get("distances") else 0.0,
                metadata=results["metadatas"][0][i] if results.get("metadatas") else {},
            ))
        return items

    def delete(self, ids: List[str]) -> None:
        self._collection.delete(ids=ids)

    def delete_by_metadata(self, filter_conditions: Dict[str, Any]) -> int:
        # ChromaDB 不直接支持按 metadata 删除，需要先查再删
        results = self._collection.get(where=filter_conditions)
        if results["ids"]:
            self._collection.delete(ids=results["ids"])
        return len(results["ids"])

    def count(self, filter_conditions: Optional[Dict[str, Any]] = None) -> int:
        return self._collection.count()
```

```python
class DeepSeekLLM(LLMBackend):
    """DeepSeek API 实现"""

                 base_url: str = "https://api.deepseek.com"):

        self._model = model

                       max_tokens: int = 2048) -> str:

            model=self._model,
            messages=messages,
            temperature=temperature,
            max_tokens=max_tokens,
            stream=False,
        )
        return resp.choices[0].message.content or ""

    async def generate_stream(self, messages: List[Dict[str, str]],
                              temperature: float = 0.7,
                              max_tokens: int = 2048) -> AsyncIterator[str]:
        stream = await self._client.chat.completions.create(
            model=self._model,
            messages=messages,
            temperature=temperature,
            max_tokens=max_tokens,
            stream=True,
        )
        async for chunk in stream:
            if chunk.choices and chunk.choices[0].delta.content:
                yield chunk.choices[0].delta.content

    @property
    def model_name(self) -> str:
        return f"deepseek/{self._model}"
```

```python
# rag/pipeline.py

from app.rag.strategies import StrategyRouter
from app.rag.interfaces.embedding import EmbeddingBackend
from app.rag.interfaces.vector_store import VectorStore

class DocumentPipeline:
    """文档处理管道：解析 → 切片 → Embedding → 存储"""

    def __init__(
        self,
        strategy_router: StrategyRouter,
        embedding: EmbeddingBackend,
        vector_store: VectorStore,
    ):
        self._strategy_router = strategy_router
        self._embedding = embedding
        self._vector_store = vector_store

    def process_document(self, file_path: str, document_id: str,
                         user_id: str, dataset_id: str = None,
                         filename: str = None) -> int:
        """
        处理单个文档的完整流程。
        返回: 生成的切片数量。
        异常:
        - ValueError: 不支持的文件格式
        - ParseError: 解析失败
        """
        # Step 1: 选择策略（parser + splitter 配对）
        ext = os.path.splitext(file_path)[1].lower()
        strategy = self._strategy_router.get_strategy(ext)
        if strategy is None:
            raise ValueError(f"不支持的文件格式: {ext}")

        # Step 2: 执行策略（解析 + 切片）
        doc_metadata = {
            "document_id": document_id,
            "user_id": user_id,
            "dataset_id": dataset_id or "__all__",
            "source": file_path,
            "document_name": filename or os.path.basename(file_path),
        }
        chunks = strategy.execute(file_path, metadata=doc_metadata)

        if not chunks:
            return 0

        # Step 3: 生成 Embedding
        texts = [chunk.content for chunk in chunks]
        vectors = self._embedding.embed_documents(texts)

        # Step 4: 存入向量库
        ids = [f"{document_id}_{chunk.chunk_index}" for chunk in chunks]
        metadatas = []
        for chunk in chunks:
            meta = dict(chunk.metadata)
            meta["chunk_index"] = chunk.chunk_index
            meta["content"] = chunk.content
            metadatas.append(meta)

        self._vector_store.add_embeddings(ids, vectors, metadatas, documents=texts)

        return len(chunks)
```

**变更说明（v1 → v2）**：
- v1 使用 `parser_router` + `splitter` 两个独立组件，所有文档类型共用 splitter
- v2 使用 `strategy_router`，每个文档类型有独立的 (parser, splitter) 配对
- v2 中 splitter 可接收 parse 产出的 `sections`，实现结构感知切分

```python
    SYSTEM_PROMPT_TEMPLATE = """你是一个智能知识库助手。你的任务是基于提供的文档内容回答用户的问题。

回答规则：
1. 严格基于提供的文档内容回答，不要编造信息。
2. 如果文档内容不足以回答问题，请明确告知"根据提供的文档，我无法回答这个问题"。
3. 使用中文回答。
4. 引用文档内容时，在相关句子后标注来源序号，如 [1][2]。
5. 回答应简洁、准确。

以下是与用户问题相关的文档内容：
{context}
"""

    def __init__(self, retriever: Retriever, llm: LLMBackend):
        self._retriever = retriever
        self._llm = llm

    async def query(self, question: str, history: Optional[List[Dict[str, str]]] = None,
                    top_k: int = 5,
                    filter_conditions: Optional[dict] = None) -> Dict[str, Any]:
        """
        执行一次完整的 RAG 查询（非流式）。
        返回: {"answer": "...", "sources": [SearchResult, ...]}
        """
        # Step 1: 检索
        results = self._retriever.retrieve(
            query=question,
            top_k=top_k,
            filter_conditions=filter_conditions,
        )

        # Step 2: 组装上下文
        context = self._format_context(results)

        # Step 3: 组装 messages
        messages = self._build_messages(question, context, history or [])

        # Step 4: 生成回答
        answer = await self._llm.generate(messages)

        # Step 5: 返回结果
        return {
            "answer": answer,
            "sources": results,
        }

    async def query_stream(self, question: str,
                           history: Optional[List[Dict[str, str]]] = None,
                           top_k: int = 5,
                           filter_conditions: Optional[dict] = None
                           ) -> AsyncIterator[Dict[str, Any]]:
        """
        流式 RAG 查询。
        每次 yield 的事件:
        - {"type": "delta", "content": "..."}  # LLM token 片段
        - {"type": "sources", "content": [...]}  # 检索结果
        - {"type": "done"}                      # 完成
        """
        # 先返回检索来源
        results = self._retriever.retrieve(
            query=question,
            top_k=top_k,
            filter_conditions=filter_conditions,
        )
        yield {"type": "sources", "content": [
            {
                "chunk_id": r.id,
                "content": r.metadata.get("content_preview", ""),
                "document_name": r.metadata.get("document_id", ""),
                "score": r.score,
            }
            for r in results
        ]}

        # 组装上下文并流式生成
        context = self._format_context(results)
        messages = self._build_messages(question, context, history or [])

        async for token in self._llm.generate_stream(messages):
            yield {"type": "delta", "content": token}

        yield {"type": "done"}

    def _format_context(self, results: List[SearchResult]) -> str:
        """将检索结果格式化为上下文文本"""
        parts = []
        for i, r in enumerate(results):
            content = r.metadata.get("content_preview", "") or ""
            parts.append(f"[{i + 1}] {content}")
        return "\n\n".join(parts)

    def _build_messages(self, question: str, context: str,
                        history: List[Dict[str, str]]) -> List[Dict[str, str]]:
        """组装 messages（含系统 prompt、历史、当前问题）"""
        system_prompt = self.SYSTEM_PROMPT_TEMPLATE.format(context=context)
        messages = [{"role": "system", "content": system_prompt}]

        # 加入历史（最多保留最近 10 轮）
        for msg in history[-20:]:  # 20 条历史消息 ≈ 10 轮对话
            messages.append({"role": msg["role"], "content": msg["content"]})

        messages.append({"role": "user", "content": question})
        return messages
```

```python
        if not self._security.verify_password(request.password, user.password_hash):
            raise ValueError("用户名或密码错误")
```

```python
                user_id=user_id,
```

```python
            await session.commit()
            await session.refresh(doc)
            doc_id = doc.id

        self._task_queue.enqueue("process_document", doc_id)
```

```python
            if result.scalar_one_or_none() is None:
                raise ValueError("对话不存在")

            query = select(Message).where(Message.conversation_id == conv_id) \
                .order_by(Message.created_at) \
                .offset((page - 1) * page_size).limit(page_size)
            messages = (await session.execute(query)).scalars().all()

            count_query = select(func.count()).select_from(Message) \
                .where(Message.conversation_id == conv_id)
            total = (await session.execute(count_query)).scalar()

        items = [await self._to_msg_response(m) for m in messages]
        return {"items": items, "total": total, "page": page, "page_size": page_size}

    async def send_message(self, conv_id: str, request: MessageSendRequest,
                           user_id: str) -> AsyncIterator[Dict[str, Any]]:
        """
        发送消息并流式获取回答。
        这是一个异步生成器，yield SSE 事件。
        """
        # 保存用户消息
        async with self._db.get_session() as session:
            result = await session.execute(
                select(Conversation).where(
                    Conversation.id == conv_id, Conversation.user_id == user_id
                )
            )
            conv = result.scalar_one_or_none()
            if conv is None:
                raise ValueError("对话不存在")

            user_msg = Message(
                conversation_id=conv_id,
                role="user",
                content=request.content,
            )
            session.add(user_msg)
            await session.commit()
            await session.refresh(user_msg)

        # 获取历史
        history = await self._get_history(conv_id)

        # 构建 filter_conditions
        filter_conditions = {"user_id": user_id}
        # 获取 conv.dataset_id
        async with self._db.get_session() as session:
            result = await session.execute(
                select(Conversation).where(Conversation.id == conv_id)
            )
            conv = result.scalar_one_or_none()
            if conv and conv.dataset_id:
                filter_conditions["dataset_id"] = conv.dataset_id

        # 流式 RAG 查询
        full_answer = ""
        sources = []
        async for event in self._rag_engine.query_stream(
            question=request.content,
            history=history,
            filter_conditions=filter_conditions,
        ):
            if event["type"] == "delta":
                full_answer += event["content"]
            elif event["type"] == "sources":
                sources = event["content"]
            yield event

        # 保存 assistant 消息
        async with self._db.get_session() as session:
            assistant_msg = Message(
                conversation_id=conv_id,
                role="assistant",
                content=full_answer,
            )
            session.add(assistant_msg)
            await session.commit()
            await session.refresh(assistant_msg)

            # 保存来源引用
            for src in sources:
                mc = MessageChunk(
                    message_id=assistant_msg.id,
                    chunk_id=src["chunk_id"],
                    relevance_score=src["score"],
                )
                session.add(mc)
            await session.commit()

            # 更新对话标题（如果是第一条消息）
            if (await session.execute(
                select(func.count()).select_from(Message)
                .where(Message.conversation_id == conv_id)
            )).scalar() == 2:  # user + assistant = 第一条回复
                conv = (await session.execute(
                    select(Conversation).where(Conversation.id == conv_id)
                )).scalar_one_or_none()
                if conv:
                    conv.title = request.content[:50] + "..." if len(request.content) > 50 else request.content
                    await session.commit()

    async def _get_history(self, conv_id: str) -> List[Dict[str, str]]:
        """获取对话历史（最近 20 条消息）"""
        async with self._db.get_session() as session:
            result = await session.execute(
                select(Message).where(Message.conversation_id == conv_id)
                .order_by(Message.created_at)
                .limit(20)
            )
            messages = result.scalars().all()
        return [
            {"role": m.role, "content": m.content}
            for m in messages
        ]

    async def _to_msg_response(self, msg: Message) -> dict:
        """将 Message ORM 转为响应 dict"""
        sources = []
        if msg.source_chunks:
            for link in await msg.source_chunks.all():
                sources.append({
                    "chunk_id": link.chunk_id,
                    "content": link.chunk.content[:200] if link.chunk else "",
                    "document_name": link.chunk.document.filename if link.chunk and link.chunk.document else "",
                    "score": link.relevance_score,
                })
        return {
            "id": msg.id,
            "role": msg.role,
            "content": msg.content,
            "sources": sources,
            "created_at": msg.created_at.isoformat() if msg.created_at else "",
        }

    def _to_conv_response(self, conv: Conversation) -> ConversationResponse:
        msg_count = conv.messages.count() if hasattr(conv, 'messages') else 0
        return ConversationResponse(
            id=conv.id,
            title=conv.title,
            dataset_id=conv.dataset_id,
            message_count=msg_count,
            created_at=conv.created_at.isoformat() if conv.created_at else "",
            updated_at=conv.updated_at.isoformat() if conv.updated_at else "",
        )
```

```python
    credentials: HTTPAuthorizationCredentials = Depends(bearer_scheme),
    container: Container = Depends(Container.get),
) -> str:
    """从 JWT 中提取当前用户 ID"""
    payload = container.security_manager.verify_token(credentials.credentials)
    if payload is None:
        raise HTTPException(
            status_code=status.HTTP_401_UNAUTHORIZED,
            detail="无效或过期的 Token",
        )

    user_id: str = Depends(get_current_user_id),
    container: Container = Depends(Container.get),

    if user.role != "admin":
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail="需要管理员权限",
        )
    return user_id
```

### 5.2 各模块测试策略速查

| 模块 | 测试类型 | Mock 策略 | 测试夹具 | 备注 |
|------|----------|-----------|----------|------|
| M01 Config | 单元 | 不需要 Mock | test.env | 纯数据验证 |
| M02 Models | 集成 | 不需要 Mock | SQLite 内存库 | ORM 关系与约束 |
| M03 Schemas | 单元 | 不需要 Mock | 构造 request 对象 | 数据校验规则 |
| M04 Database | 集成 | 不需要 Mock | SQLite 内存 URL | 连接与表创建 |
| M05 Security | 单元 | `Settings(secret="test")` | 测试用密钥/密码 | 哈希与 JWT 往返 |
| M06 Storage | 集成 | 不需要 Mock | `tempfile.TemporaryDirectory` | 文件读写生命周期 |
| M07 TaskQueue | 集成 | 不需要 Mock | 临时 .sqlite 文件 | 任务注册与执行 |
| M08 Interfaces | 单元 | Mock 实现类 | — | 接口契约 |
| M09 Parsers | 集成 | 不需要 Mock | `fixtures/sample.txt` 等 | 真实文件解析 |
| M10 Splitter | 单元 | 不需要 Mock | 字符串输入 | 分块逻辑 |
| M11 Embedding | 混合 | 模型层可 Mock | — | 维度验证用 Mock，质量测试用集成 |
| M12 VectorStore | 集成 | 不需要 Mock | `tempfile.TemporaryDirectory` | CRUD 操作 |
| M13 LLM | 单元 | Mock `httpx` / `openai` | — | 流式与非流式 |
| M14 Retriever | 单元 | Mock Embedding + VectorStore | — | 调用链验证 |
| M15 Pipeline | 单元 | Mock 所有 RAG 组件 | — | 流程编排 |
| M16 RAG Engine | 单元 | Mock Retriever + LLM | — | Prompt 组装 |
| M17 UserService | 单元 | Mock DatabaseManager + Security | 内存 DB | 注册/登录/密码 |
| M18 DatasetService | 单元 | Mock DatabaseManager | 内存 DB | CRUD + 用户隔离 |
| M19 DocumentService | 单元 | Mock 所有外部依赖 | 内存 DB | 上传/状态/删除 |

| M22 API | 集成 | Mock Container 下所有 Service | TestClient | 路由与鉴权 |

### 5.3 测试夹具设计原则

1. **每个模块自包含**：模块的 `tests/fixtures/` 目录只存放本模块测试需要的数据
2. **共享夹具放 `services/tests/fixtures/`**：仅 Service 层测试可共享 mock_db.py
3. **测试用工具函数不要跨模块引用**：如果多个模块需要相同的 mock 工具函数，放在 `tests/conftest.py` 中
4. **不做网络请求**：所有涉及外部服务（LLM API、Embedding API）的测试必须 Mock 网络层

### 5.4 测试运行命令

```bash
# 运行所有模块测试
pytest

# 运行特定模块测试
pytest app/rag/parsers/tests/
pytest app/services/tests/

# 跳过集成测试（需要外部资源的测试）
pytest -m "not integration"

# 运行带覆盖率的测试
pytest --cov=app --cov-report=term-missing
```
